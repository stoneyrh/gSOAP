Detailed Description

This file defines the common macros, types and functions of the gSOAP API grouped by modules

The API features are grouped by the following modules:

  • Debugging and logging
  • WITH_MACRO compile-time flags
  • SOAP_MACRO compile-time values
  • SOAP_MACRO run-time flags
  • SOAP_MACRO run-time error codes
  • Context with engine state
  • Callback functions
  • SSL/TLS context and functions
  • HTTP and IO functions
  • HTTP cookie functions
  • Conversion functions
  • XML namespace tables
  • Header structure and functions
  • Fault structure and functions
  • DIME attachment functions
  • MIME attachment functions
  • Plugins and plugin registry functions
  • Miscellaneous functions
