Detailed Description

Schema urn:uddi-org:api_v2:"find_relatedBusinesses".

Represents a request to locate businesses that are related to a specific business. When the send method is called, the instance returns a uddi2__relatedBusinessList object that contains a list of business that matched the relationship set.

See http://uddi.org/pubs/ProgrammersAPI_v2.htm#_Toc25137714
