Detailed Description

Schema urn:uddi-org:api_v2:"discard_authToken".

Represents a request to discard an authorization token obtained with the get_USCOREauthToken class.

See http://uddi.org/pubs/ProgrammersAPI_v2.htm#_Toc25137737
